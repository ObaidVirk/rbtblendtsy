# rbtblendtsy.com Frontend

A production-ready frontend built with Next.js 14, TypeScript, and Tailwind CSS for a modern banking and fintech experience.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## Features

- UI clone with branded pages and sections
- Login and signup flows
- Admin dashboard
- Language selector with 20 languages

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

## Environment Variables

Create a local `.env.local` file (do not commit it) and use `.env.example` as the template. Store all sensitive values there, including admin credentials and Stripe keys.

## Build

```bash
npm run build
```
